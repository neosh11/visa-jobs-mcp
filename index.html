Visa Jobs MCP

Always free. Completely private. Built for agents.

Find relevant jobs from fresh LinkedIn searches with optional visa filtering when needed. Use local-first workflows for save/apply/ignore without sharing user data externally.

Install on macOS (Homebrew)

brew tap neosh11/visa-jobs-mcp
brew install neosh11/visa-jobs-mcp/visa-jobs-mcp

Install on macOS/Linux (bash)

curl -fsSL https://raw.githubusercontent.com/neosh11/visa-jobs-mcp/main/scripts/install.sh | bash

Install on Windows (PowerShell)

irm https://raw.githubusercontent.com/neosh11/visa-jobs-mcp/main/scripts/install.ps1 | iex

Homebrew tap: github.com/neosh11/homebrew-visa-jobs-mcp
Current release: v0.3.1

Configure in Codex (macOS/Linux)

codex mcp add visa-jobs-mcp --env VISA_JOB_SITES=linkedin -- /opt/homebrew/bin/visa-jobs-mcp
codex mcp list
codex mcp get visa-jobs-mcp

Configure in Codex (Windows)

codex mcp add visa-jobs-mcp --env VISA_JOB_SITES=linkedin -- "$env:LOCALAPPDATA\\Programs\\visa-jobs-mcp\\bin\\visa-jobs-mcp.exe"
codex mcp list
codex mcp get visa-jobs-mcp

Then ask your agent:

  • Find IT jobs in Delhi relevant to my skills.
  • Set my visa preference to E3 and find software engineer jobs in New York that sponsor E3.

How It Works

  1. Search fresh jobs by location + title.
  2. Filter locally for role relevance and ignored jobs/companies.
  3. Optionally apply visa matching if preferences are set.
  4. Return links, contacts, eligibility reasons, and confidence.

What You Get

  • LinkedIn-focused job discovery.
  • General search with no visa setup required.
  • Optional visa preference filtering when configured.
  • Session-based pagination and resume behavior.
  • Saved jobs and ignored jobs.
  • Local user preferences and memory blob.
  • No proxies. No LLM runtime inside MCP.
